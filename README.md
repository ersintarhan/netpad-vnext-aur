# netpad-vnext-bin

NetPad vNext - A cross-platform C# editor and playground with native Rust-based shell.

## Description

This package provides the vNext version of NetPad, which uses a native Rust-based shell (Tauri) instead of Electron, offering better performance and lower resource usage.

## Features

- Native Rust-based UI (lighter than Electron version)
- C# scripting and LINQ queries
- NuGet package support
- Database connectivity
- .NET 9 support
- Faster dump/display performance

## Installation

```bash
paru -S netpad-vnext-bin
# or
yay -S netpad-vnext-bin
```

## Usage

```bash
netpad-vnext
```
